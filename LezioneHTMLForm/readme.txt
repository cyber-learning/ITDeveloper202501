# Link Esterni e Link Interni
1) Creare uno o più link esterni.
2) Creare uno o più link interni (al sito stesso).
3) Creare una directory contenente pagine html (ad esempio users contenente le pagine degli utenti)
    aggiungendo i link alle pagine presenti nella directory




Elementi Block:
- div (contenitore degli elementi block)
- headings tag h1 - h6
- p
- ul/ol li 
unordered list - list item
ordered list - list item
- form

Elementi Inline:
- span (contenitore degli elementi inline)
- a
- strong
- input (self closing tag)
- label
- button

Attributi globali:
- id identificare univocamente uno e solo un elemento
- class (sinonimo di insieme)
- style
- onclick onsubmit


Tag semantici: elementi HTML con un "ruolo" ben specifico (alla fine sono dei div)
- nav
- header presenta del contenuto introduttivo per l'elemento parent
- footer presenta del contenuto secondario per l'elemento parent
- main, article, section
- aside







# input
type:
- text
- number
- email
- password

- submit (strano)





Di default, al submit della form il browser:
1) impacchetta i dati presenti negli elementi input in un certo formato (application/x-www-form-urlencoded)
2) invia i dati al server

Una query string può essere utilizzata solo con il metodo http GET al fine di filtrare i risultati di ricerca



# Esercizio - Blog di ricette
Prendendo ispirazione da un blog di ricette già esistente (ad esempio Giallo Zafferano), strutturare la pagina index.html utilizzando tutto ciò che abbiamo visto in HTML ed in particolare:
- tag semantici 
- headings tag (sia article sia section devono presentare un heading tag)

Per le immagini utilizzare picsum.photos modificando opportunamente l'URL dell'immagine:
https://picsum.photos/id/1/400/300

Nella navbar aggiungere un link interno che rimandi ad una pagina di iscrizione come chef.
Questa pagina deve presentare una form che richieda le seguenti informazioni:
- nome e cognome
- età
- email
- password
- chef stellato?
- accetti termini e condizioni?
